# C++线程池

基于可变参模板和现代C++17标准实现的高性能线程池，支持固定模式（fixed）和可扩容模式（cached），适用于高并发任务处理场景。

## 特性

**两种工作模式**

- MODE_FIXED：固定线程数量
- MODE_CACHED：动态扩容线程

**现代C++特性**

- 可变参模板：支持任意参数的任务提交
- std::future：异步获取任务结果
- RAII资源管理：智能指针自动管理线程资源

**高性能设计**

- 任务队列容量可配置
- 线程数量阈值可配置
- 避免频繁创建/销毁线程的开销

**线程安全**

- `mutex`+`condition_variable`保证线程同步
- 支持生产者-消费者模式

## 环境要求

编译器：支持C++17

操作系统：Windows/Linux
